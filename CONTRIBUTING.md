# Contributing to Arduino-Beginner-Master-Hub 👋

First off, thank you for considering contributing to the Master Hub! This repository was built specifically to help absolute beginners get into electronics and coding. 

We absolutely **love** first-time contributors! If you have never made a Pull Request on GitHub before, this repository is the perfect, safe place to practice.

## How to Add a New Project 🚀

If you want to add "Project 25" (or whatever the next number is) to the Hub, please follow these strict rules to keep the repository organized:

### 1. Folder Structure Rules
- Every new project must go into the `projects/` directory.
- The folder must be numbered sequentially based on the last project (e.g., `25_your_project_name`).
- The Arduino code file inside the folder **must** perfectly match the folder name (e.g., `25_your_project_name.ino`).

### 2. Code Quality Rules
- The code must be heavily commented so a beginner can understand exactly what every line does.
- Please review our **[Clean Code Guide](./docs/clean_code_guide.md)** before submitting your `.ino` file.

### 3. The `README.md` Template
Every project folder must contain a `README.md` file that follows our exact standard format. Copy and paste the template below into your new project, and fill in the blanks!

```markdown
# Project XX: [Your Project Title] [Relevant Emoji]

[Write a 2-3 sentence introduction explaining what the component is and what this project does in simple terms.]

## 📦 Component List
- 1x Arduino Uno
- [List your components here]
- Jumper Wires

---

## 🔌 Circuit Connection Guide

**[Component Name]:**
- `Pin Name` -> Connect to **Arduino Pin**.

---

## 🧠 Code Explanation

- `[Code snippet]`
  [Explain what this function does in plain English].

---

## 🌐 Tinkercad Simulation

Don't have the physical components yet? No problem! You can build and test this exact circuit in your web browser before touching real hardware.
[Click here to open Tinkercad](https://www.tinkercad.com/things/YOUR_UNIQUE_LINK) to build this project virtually!
```

## How to Submit Your Project 📤
1. **Fork** this repository to your own GitHub account.
2. **Clone** it to your computer and add your new project folder.
3. **Commit** your changes with a descriptive message (e.g., `feat: added Project 25 Water Pump`).
4. **Push** your branch up to GitHub.
5. Open a **Pull Request** on our main repository. Make sure to fill out the checklist in the PR template!

We can't wait to see what you build!
