# Components Glossary

- PIR Sensor: Motion detection
- Sound Sensor: Audio levels
- Rotary Encoder: Rotation input
- Keypad: 4x4 buttons
- Relay: Switch for high power
- RTC DS3231: Timekeeping
- SD Card Module: Storage
- Soil Moisture: Water level
- Flame Sensor: Fire detection
- Pulse Sensor: Heart rate
