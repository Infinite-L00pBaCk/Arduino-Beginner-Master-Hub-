# 09 — Mini Projects 🚀

Two complete mini-projects that combine multiple concepts from earlier lessons.

---

## Projects in this folder

### [Traffic Light Simulator](./traffic_light/)
Simulates a RED → GREEN → YELLOW traffic light sequence using 3 LEDs.
**Concepts:** Multiple digital outputs, timing sequences, functions

### [Digital Clock](./digital_clock/)
A working clock on a 16x2 LCD that counts hours, minutes, and seconds.
**Concepts:** LCD display, `millis()` non-blocking timer, string formatting

---

These projects are great starting points — extend them with the "Try It Yourself" challenges in each folder's README!
