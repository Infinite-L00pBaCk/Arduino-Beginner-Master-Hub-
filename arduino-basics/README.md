## 🛠️ Getting Started

### What You Need

**Hardware:**
- Arduino Uno (or Nano / Mega)
- USB cable (Type A to Type B)
- Breadboard + jumper wires
- Basic components (LEDs, resistors, buttons, sensors — listed per project)

**Software:**
- [Arduino IDE](https://www.arduino.cc/en/software) — download and install
- Required libraries (listed per project)

### How to Upload Code

```
1. Connect your Arduino to your PC via USB
2. Open Arduino IDE
3. Go to File → Open → select the .ino file
4. Tools → Board → Select "Arduino Uno"
5. Tools → Port → Select the correct COM port
6. Click the Upload button (→ arrow icon)
7. Open Serial Monitor (Ctrl + Shift + M) if needed
```

## 📌 Arduino Pin Reference (Uno)

```
Digital Pins  : D0 – D13
Analog Pins   : A0 – A5
PWM Pins      : D3, D5, D6, D9, D10, D11
Power Pins    : 5V, 3.3V, GND
Serial (UART) : D0 (RX), D1 (TX)
I2C           : A4 (SDA), A5 (SCL)
SPI           : D10 (SS), D11 (MOSI), D12 (MISO), D13 (SCK)
```
